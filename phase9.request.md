자잘한 UI 개선사항 두 가지를 반영해줘.

1. 이동 버튼 아이콘 통일
   - 홈 화면으로 돌아가는 버튼과, 상세 페이지에서 
     이전 목록으로 돌아가는 "이전" 버튼을 
     기존 다크모드 토글·로그아웃 버튼과 동일한 스타일의 
     아이콘 버튼으로 통일해줘 (lucide-react 아이콘 사용)
   - 적용 범위: 보유종목 상세, 관심종목 상세, 
     지수별 상세 페이지 등 "이전/홈" 버튼이 있는 모든 곳

2. 종목 추가 폼을 토글 방식으로 변경
   - 보유종목 상세(/holdings)와 관심종목 상세(/watchlist) 
     페이지의 "종목 추가" 입력 영역이 지금은 항상 펼쳐진 
     상태로 보이는데, 이걸 "추가" 버튼을 눌러야만 
     입력 폼이 나타나는 방식으로 바꿔줘
   - 평소에는 "+ 종목 추가" 같은 버튼만 보이고, 
     클릭하면 그 자리에 입력 폼이 펼쳐지는 구조

둘 다 사소한 UI 조정이니 코드 바로 작성해도 돼.
