# Phase 16 추가 수정 — 수익률/시작종가/끝종가 값 오른쪽 정렬

## 배경
전체 항목명·항목값이 가운데 정렬로 통일되어 있는데, 그중 수익률/
시작종가/끝종가는 숫자 데이터이므로 값만 오른쪽 정렬로 변경한다.

## 요구사항

1. **대상 열**: 수익률, 시작종가, 끝종가 3개 열
2. **항목명(th)**: 기존 가운데 정렬 그대로 유지 (변경 없음)
3. **항목값(td)**: 오른쪽 정렬로 변경
   - 이 3개 열의 값만 text-align: right 적용
   - 그 외 열(순위, 종목명, 종목코드 등)의 정렬은 이번 변경과
     무관하게 기존 그대로 둔다
4. 기존 폰트 크기·색상·padding 등 다른 스타일 속성은 변경하지 않는다

## 완료 조건
- 수익률/시작종가/끝종가 값만 오른쪽 정렬
- 항목명 정렬은 변경 없음
- 다른 열 정렬·스타일 변경 없음
- lint·tsc·build 통과
